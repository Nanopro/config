# Tab Navigation

|Key   |What   |Additional   |
|---|---|---|
|`,tn`   | Create new tab  |   |
|`,tc`  | Close current tab  |   |
| `,t,`  | Next Tab  |   | 
| `,tm` | Move Tab | Requires number|
| `,to` | Close All other Tabs except current |


# Window Navigation

| Key  |What   | Additional  |
|---|---|---|
| `C-j`  | Move focus to the bottom  |   |
| `C-k`  | Move focus to the top  |   |
| `C-l`  | Move focus to the left  |   |
| `C-h`  | Move focus to the right  |   |
